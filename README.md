# MinoCV - 免费在线简历生成器

> 无需注册，打开即用的简历生成器。20+ 精美主题，AI 智能优化，一键导出 PDF。

## ✨ 功能特点

- 🎨 **20+ 精美主题** - 从简约到创意，总有一款适合你
- 🖱️ **拖拽排序** - 自由调整简历章节顺序
- 👀 **实时预览** - 左边编辑，右边立即看到效果
- 🤖 **AI 智能优化** - 根据职位描述自动优化简历内容
- 📄 **多格式导出** - 支持 PDF、HTML、JSON 导出
- 💾 **本地存储** - 数据自动保存在浏览器，无需登录

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 🛠️ 技术栈

- React + TypeScript
- Vite
- Tailwind CSS
- Radix UI
- OpenAI API (AI 优化功能)

## 📝 使用说明

1. 在左侧面板填写个人信息
2. 选择喜欢的主题
3. 使用 AI 功能优化简历（可选）
4. 导出 PDF 或 HTML
